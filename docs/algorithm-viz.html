Reuse pattern
DataWhereReused how
acc[K] registers Persists across all N_parts iterations. Never written to smem or global until the final write-out. Full register reuse.
smem_A shared mem Overwritten each partition. 53 A-diagonal chunks reloaded from global A_vals. The same smem region is reused — no extra allocation.
smem_B shared mem Overwritten each partition. ≤60 B-diagonal chunks from global B_vals. Determined by PartBMeta[p]: d_b = d_c − d_a maps A partition → B range.
lookup shared mem Rebuilt from PartBMeta each partition (O(b_count) inits). Maps relative d_b → smem slot index for O(1) B access in inner loop.
Why O(1) smem in N:
Total A diags per group grows O(N). But each CTA holds only 53 at a time. acc[] carries the running sum in registers — no global partial buffer. Smem footprint is the same whether N = 1K or N = 16K.
d_b contiguity (balanced):
A sorted by d_a ascending → partition of 53 A diags spans [d_a_min, d_a_max]. Then d_b = d_c − d_a spans [d_c_min−d_a_max, d_c_max−d_a_min], range = (K−1) + 53 = 60 entries. B partition is always contiguous.
